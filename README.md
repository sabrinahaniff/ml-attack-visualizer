# ML Attack Visualizer

I built this for myself. I've been exploring AI security independently and I kept running into the same problem which is: I could understand the concepts when someone explained them, but I couldn't see what was actually happening. Like for example; What does it look like when you nudge a pixel? What happens to federated learning when you add one more malicious client?

So I built this to make it visual and interactive.

I don't have a strong math background in ML. I'm learning this outside of class out of curiosity, and this is how I process things by building something I can actually play with.

Covers my four projects so far:
- Adversarial image attacks (FGSM)
- Federated learning poisoning
- LLM red teaming
- Membership inference attacks

## Setup

```bash
npm install
npm run dev
```
