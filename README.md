# ML Attack Visualizer

Recently, I've been exploring AI security independently and I kept running into the same problem which is: I could understand the concepts when someone explained them, but I couldn't see what was actually happening. Like for example; What does it look like when you nudge a pixel? What happens to federated learning when you add one more malicious client?

And so, I built this to make it visual and interactive mainly for my self learning.

![alt text](image.png)

Covers my four projects so far:
- Adversarial image attacks (FGSM)
- Federated learning poisoning
- LLM red teaming
- Membership inference attacks
